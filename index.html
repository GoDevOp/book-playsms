Howto read the book locally

This project will include information about how to install, configure and use playSMS. This project will also host several related open documents and other notes related to playSMS.

playSMS is a free and open source SMS management software.

A flexible Web-based mobile portal system that it can be made to fit to various services such as an SMS gateway, bulk SMS provider, personal messaging system, corporate and group communication tools.

Visit playSMS Project website for source codes and more information about playSMS.
